# 📋 To-Do List App

Aplikasi to-do list berbasis web yang bisa diakses dari mana saja — tanpa install, cukup buka browser.

🔗 **Live Demo:** `https://<username>.github.io/todo-app`

---

## ✨ Fitur

- ✅ Tambah, edit, hapus tugas
- 📂 Kategori: Work, Personal, Wishlist, Birthday
- 📅 Tampilan Kalender bulanan
- 📊 Profil & statistik produktivitas
- 🔖 Tandai tugas dengan simbol (bendera, angka, progres, mood)
- ⭐ Bintang / prioritas tugas
- 🌙 Dark mode
- 💾 Data tersimpan di browser (localStorage)
- 📱 Responsif mobile & desktop

## 🚀 Cara Pakai

Buka langsung via GitHub Pages — tidak perlu install apapun.

## 🛠 Teknologi

- HTML5 / CSS3 / Vanilla JavaScript
- LocalStorage untuk penyimpanan data
- Google Fonts (Nunito + Poppins)
- Zero dependencies — satu file HTML
